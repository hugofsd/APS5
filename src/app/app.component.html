MONITORAMENTO DE LIQUEFAÇÃO:

O trabalho teve como motivação apresentar uma solução para prevenir rompimentos
de barragens, para assim evitar desastres, como aconteceu em Brumadinho-MG.
Para tanto utilizamos um Arduino, com sensor de umidade, para que ele pudesse
detectá-la nas barragens, e quando um nível alto fosse atingido, enviaria
um alerta. Além disso, uma interface HTML foi
criada para o controle dos dados obtidos.

• Monitoramento por região
• Levantamento semanal
• Baixo custo sistemico

• Arduino com Cabo USB
• Senso de Umidade do Solo
• Cabos Jumper Marcho-Fêmea
• Cabos Jumper Fêmea-Fêmea
• Cabo jumper macho-fêmea da porta A0(Sensor) para porta A0(Arduino)
• Cabo jumper macho-fêmea da porta GND(Sensor) para porta GND(Arduino)
• Cabo jumper macho-fêmea da porta VCC(Sensor) para porta 5V(Arduino)
• Cabo jumper fêmea-fêmea da porta +(Sensor) para Higrômetro
• Cabo jumper fêmea-fêmea da porta –(Sensor) para Higrômetro
